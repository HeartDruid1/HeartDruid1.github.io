Basic Python Twitter Bot

This Twitter bot written in Python 3 tweets a message made up of randomly selected words from a .csv file (which can be written in Microsoft Excel or any other spreadsheet software) and tweets the message at 15 minute intervals with a handy countdown clock that counts every minute in the command line and when the message tweets the command line will print it in the command line. *Due to issues with Twython (which is a required library) the script crashes after an hour* Script Word List

Minecraft Pi Chat GUI

This one is a little different and requires a Raspberry Pi, Minecraft Pi, and the guizero library for Python3.5.2. This is a very simple and basic script that demonstrates how Python and GUIs can be used in unison with additional software. It works like so, you click on the text box and type the message you wish to post to the chat board and press the provided push button to post. The script may also be easily modified to run with Twython and be used as a custom Twitter application. It's a piece of cake... or rather "Pi" ;) Click Here
