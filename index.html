My Toolbox

Since it's a huge subject to controversy, I figured I'd make a list of the primary tools I use regularly while programming

  • Vim Text Editor (Emacs sucks ;) )
  • Linux Mint (although considering switching to Arch in the future... Arch Linux Master Race!)
  • GitKraken/Git Cola as git clients
  • SeaMonkey web browser (built in tools for web development and it sounds cool)
  • GIMP & ASEPRITE for drawing and photo editing tools
  • Clementine Media Player (for intense concentration tunes)

Houston, we have git!

As I'm running 32 bit Linux Mint, my favourite git client is not an option (GitKraken) and I haven't been able to find one that I really liked, until now. I found 'git cola' and so far it seems to do exactly what GitKraken is able to. This is actually a test post to see the capabilities of 'git cola'. If it works out well I might just have found a new git client ;)
